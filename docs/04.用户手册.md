# 04.用户手册

## 1. 环境要求

- 安装 Node.js 18 或以上版本

## 2. 启动步骤

1. 打开项目目录
2. 执行 `npm start`
3. 浏览器访问 `http://localhost:3000`

## 3. 角色操作说明

### 3.1 管理员

- 登录后可在左侧菜单进入学生管理、教师管理、班级/院系管理、系统日志、数据概览
- 点击新增按钮可录入基础数据
- 编辑用户可修改姓名、班级、院系等信息

### 3.2 教师

- 进入题库管理创建题目
- 进入试卷管理选择题目组卷
- 进入考试管理发布考试
- 学生交卷后，在阅卷中心可给简答题打分
- 在成绩分析查看整体数据

### 3.3 学生

- 在待考考试中进入考试
- 作答过程中可切换题号并点击保存
- 提交后可在考试记录查看成绩
- 错题本自动展示答错题目
- 个人信息中可修改密码

## 4. 常见问题

- 页面打不开：确认 `npm start` 已成功启动
- 登录失败：确认使用演示账号
- 修改密码后忘记：可手动编辑 `data/store.json` 重置
