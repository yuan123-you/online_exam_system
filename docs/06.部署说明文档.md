# 06.部署说明文档

## 1. 环境信息

| 项目 | 版本 | 说明 |
|------|------|------|
| Java | JDK 21+ | Spring Boot 后端运行环境 |
| Node.js | 18+ | 前端构建和脚本运行 |
| MySQL | 8.x | 数据库 |
| Maven | 3.x | 后端构建工具 |
| 操作系统 | Windows / macOS / Linux | 跨平台支持 |

## 2. 完整部署步骤

### 2.1 克隆项目

```bash
cd "D:\Codex Web\20252160A1025游源"
```

### 2.2 初始化数据库

```bash
mysql -u root -p < database.sql
```

该脚本会自动创建 `online_exam_system` 数据库，建立 9 张数据表，并导入演示数据（包括 1000+ 道题目、11 套试卷、4 场考试等）。

### 2.3 配置数据库连接

默认配置位于 `backend/src/main/resources/application.yml`：

```yaml
MYSQL_URL: jdbc:mysql://localhost:3306/online_exam_system?createDatabaseIfNotExist=true&useSSL=false&serverTimezone=Asia/Shanghai
MYSQL_USER: root
MYSQL_PASSWORD: 123456
```

如果本机 MySQL 密码不同，可通过环境变量覆盖：

```powershell
# Windows PowerShell
$env:MYSQL_USER="root"
$env:MYSQL_PASSWORD="你的密码"
```

```bash
# Linux / macOS
export MYSQL_USER=root
export MYSQL_PASSWORD="你的密码"
```

### 2.4 构建并启动后端

```bash
# 编译后端
npm run backend:build

# 启动后端
npm start
```

或手动运行 JAR：

```bash
java -jar backend/target/online-exam-backend-1.0.0.jar
```

后端启动成功后访问 `http://localhost:8080/api/health` 验证。

### 2.5 启动前端（Vue 3 版本）

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev:web
```

前端启动后访问 `http://localhost:5173`，Vite 会自动将 `/api` 请求代理到后端 8080 端口。

### 2.6 使用独立版（无需前端构建）

Spring Boot 启动后直接访问 `http://localhost:8080` 即可使用内置的原生 HTML 版本。

## 3. 构建生产版本

```bash
# 构建前端生产版本
npm run build:web

# 输出目录：dist/

# 预览生产版本
npm run preview:web
```

## 7. 注意事项

- 首次启动后端会自动执行 `schema.sql` 和 `data.sql` 初始化数据库
- 若 8080 端口被占用，可在 `application.yml` 中修改 `server.port`
- 若 5173 端口被占用，可在 `vite.config.ts` 中修改 `server.port`
- 独立版数据存储在 `data/store.json`，与 MySQL 数据独立
- 生产部署时建议将前端构建产物（`dist/`）部署到 Nginx，后端部署为独立 JAR
