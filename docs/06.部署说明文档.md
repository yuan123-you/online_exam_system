# 06.部署说明文档

## 1. 环境信息

- Node.js：18+
- 操作系统：Windows / macOS / Linux

## 2. 本地部署步骤

1. 进入项目目录：`D:\Codex Web\online exam system`
2. 安装依赖：`npm install`
3. 启动项目：`npm start`
4. 浏览器访问：`http://localhost:3000`

## 3. 项目运行命令

```bash
npm install
npm start
```

## 4. 演示账号

- 管理员：`admin / 123456`
- 教师：`teacher / 123456`
- 学生：`2023001 / 123456`

## 5. 目录说明

- `server.js`：后端入口
- `public/`：前端页面资源
- `data/store.json`：运行期数据
- `database.sql`：数据库脚本
- `docs/`：结题文档资料

## 6. 注意事项

- 首次运行会自动生成 `data/store.json`
- 若端口 3000 被占用，可结束占用进程后重启
