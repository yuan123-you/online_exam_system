## 4. 性能与稳定性说明

- Spring Boot + MySQL 架构可稳定支撑课程项目演示规模
- 前端 Vite 开发服务器热更新快速，开发体验良好
- ECharts 图表在数据量合理范围内渲染流畅
- 因采用单节点部署，不适合作为高并发生产系统
